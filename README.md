# PROMTP
Necesito crear una pantalla en Flutter usando el lenguaje Dart. El código debe estar todo en un solo archivo llamado main.dart y debe ser sencillo, pensado para nivel principiante.
La interfaz debe organizarse usando filas (Row) y columnas (Column).
Primero debe haber una Column principal que contenga todos los elementos de la pantalla.
El fondo de la pantalla debe tener un degradado de color azul que se vaya desvaneciendo hacia blanco, comenzando más oscuro en la parte superior y más claro hacia la parte inferior.
En la parte superior de la pantalla debe haber una Row con dos elementos alineados horizontalmente:
un icono de flecha hacia atrás
un texto que diga “DetalleGasto” con tamaño de letra grande.
Debajo del título debe haber una Column que contenga cuatro campos de texto (TextField) organizados verticalmente. Cada campo debe tener espacio entre ellos. Los campos deben ser:
Monto
Categoria
Descripcion
Fecha
Los campos deben verse como cajas blancas con borde.
Después de los campos debe haber otra Column con dos botones grandes centrados:
El primer botón debe ser de color verde con el texto “Actualizar Gasto”.
Debajo debe haber otro botón de color rojo con el texto “Eliminar Gasto”.
Los botones deben tener espacio entre ellos y el texto debe estar centrado.
En la parte inferior de la pantalla debe existir una barra de navegación inferior (BottomNavigationBar) con cuatro opciones organizadas horizontalmente:
Inicio con icono de casa
Gastos con icono de dinero
Ingresos con icono de dinero hacia arriba
Perfil con icono de usuario
La opción Gastos debe aparecer seleccionada.
El código debe incluir comentarios simples explicando la estructura de filas y columnas, y debe usar widgets básicos de Flutter como:
Scaffold
Container
Row
Column
TextField
ElevatedButton
BottomNavigationBar
Icon
fragmentos incompletos
